# V1SearchController Implementation Spec

## Summary

- Problem: `parseFacetFields(...)` uses `facetFields.size() > 0` to test whether a list is non-empty, and Sonar flags that pattern under `java:S1155`.
- Goal: Replace the size comparison with `!facetFields.isEmpty()` while preserving the existing `facetFields != null` guard and surrounding loop behavior.
- In scope: Update only the non-empty collection check at the flagged location.
- Out of scope: Refactoring the rest of `parseFacetFields(...)`, changing null handling, or altering facet parsing behavior.
