# V1SearchController `defType` Constant Extraction Spec

## Summary

- Problem: `V1SearchController` repeats the Solr parameter key `defType` in multiple `solrQuery.set(...)` calls, producing the duplication tracked by `SG-BLG-00016`.
- Goal: Replace the repeated key with one private constant while keeping every affected query call identical in effect.
- In scope: The `defType` parameter-name literal inside `V1SearchController`.
- Out of scope: Changing the associated parser value, editing other request parameter names, or altering query semantics.
